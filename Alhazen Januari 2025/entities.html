Entities HTML

Entities HTML adalah karakter yang memiliki makna khusus dalam HTML. Entities HTML digunakan untuk menampilkan karakter khusus yang tidak bisa ditampilkan dengan mudah dalam HTML.

Entities HTML ditulis dengan menggunakan karakter & diikuti dengan nama karakter dan diakhiri dengan tanda titik koma (;).

Berikut adalah beberapa contoh Entities HTML:

Entities HTML Character Description
&nbsp; Non-Breaking Space
&lt; < Less-Than Sign
&gt; > Greater-Than Sign
&copy; © Copyright Sign
&reg; ® Registered Sign
&trade; ™ Trademark Sign

Entities HTML sangat berguna untuk menampilkan karakter khusus dalam HTML. Entities HTML juga dapat digunakan untuk menampilkan karakter yang tidak bisa ditampilkan dengan mudah dalam HTML.

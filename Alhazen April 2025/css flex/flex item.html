Mengatur tata letak dengan flex juga dapat dilakukan di level flex item. Beberapa properti yang dapat kita gunakan di antaranya

order, mengatur urutan flex item
flex-grow, mengatur ukuran lebar flex item relatif terhadap item lainnya
flex-shrink, mengatur ukuran penyusutan flex item relatif terhadap item lainnya
flex-basis, mengatur ukuran awal untuk flex item
align-self, mengatur alignment flex item secara satuan

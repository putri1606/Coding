Belajar HTML

HTML adalah bahasa markup yang digunakan untuk membuat dan menyusun konten di web. Dengan HTML, pengguna dapat menentukan struktur halaman web, termasuk elemen seperti teks, gambar, dan tautan.

Belajar CSS

CSS adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web. Dengan CSS, pengembang dapat mengontrol elemen seperti warna, font, dan tata letak.

belajar javascript

javascript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif dan dinamis. javascript dapat digunakan untuk memanipulasi elemen HTML, mengubah gaya CSS, dan menangani peristiwa pengguna seperti klik, ketikan, dan gerakan mouse.


alert adalah fungsi untuk menampilkan pesan ke pengguna atau POP UP
prompt adalah fungsi untuk meminta input dari pengguna melalui kotak dialog. Kotak dialog ini menampilkan pesan dan sebuah kolom input untuk pengguna memasukkan nilai. Setelah pengguna mengisi kolom input dan menekan tombol OK, nilai yang dimasukkan akan dikembalikan sebagai string. Jika pengguna menekan tombol Cancel, maka fungsi prompt akan mengembalikan nilai null.


variabel adalah tempat untuk menyimpan data atau nilai dalam javascript. Ada beberapa jenis variabel dalam javascript, yaitu:

  • var
    • var adalah cara lama untuk mendeklarasikan variabel. Variabel yang dideklarasikan dengan var memiliki cakupan fungsi atau global, tergantung di mana mereka dideklarasikan.
    • var dapat dideklarasikan ulang dan diinisialisasi kembali.
  • let
    • let adalah cara baru untuk mendeklarasikan variabel. Variabel yang dideklarasikan dengan let memiliki cakupan blok, artinya hanya dapat diakses di dalam blok tempat mereka dideklarasikan.
    • let tidak dapat dideklarasikan ulang, tetapi dapat diinisialisasi kembali.
  • const
    • const adalah cara untuk mendeklarasikan variabel yang nilainya tidak dapat diubah setelah dideklarasikan. Variabel yang dideklarasikan dengan const juga memiliki cakupan blok.
    • const tidak dapat dideklarasikan ulang dan tidak dapat diinisialisasi kembali.
